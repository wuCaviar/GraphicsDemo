## Algorithm Description

### Naive (current `color_dialog.cpp`)

```text
K = 1 - max(R, G, B)          // R,G,B in 0.0-1.0
C = (1 - R - K) / (1 - K)
M = (1 - G - K) / (1 - K)
Y = (1 - B - K) / (1 - K)
```

### Photoshop-like

```text
// Step 1: sRGB gamma decode
if (V <= 0.04045) V_lin = V / 12.92
else              V_lin = ((V + 0.055) / 1.055) ^ 2.4

// Step 2: Invert to CMY
C_raw = 1 - R_lin;  M_raw = 1 - G_lin;  Y_raw = 1 - B_lin

// Step 3: Black generation (GCR)
K = min(C_raw, M_raw, Y_raw) * gcrFactor

// Step 4: Adjust CMY
C = (C_raw - K) / (1 - K)
M = (M_raw - K) / (1 - K)
Y = (Y_raw - K) / (1 - K)

// Step 5: Total ink limit (default 300%)
if (C + M + Y + K > 3.0) { scale = 3.0 / total; C*=s; M*=s; Y*=s; K*=s }
```
